**พิสูจน์ตัวตนของคุณ** — แนะนำตัวเองให้แก๊งรู้จัก

## ขั้นตอนปฏิบัติการ

### 1. รับแฟ้มภารกิจ

- เข้าถึงแฟ้มลับที่ส่งมาให้คุณ
- คัดลอกไว้ในฐานปฏิบัติการส่วนตัวของตัวเอง โดยทำตามขั้นตอนนี้:

**1.1 Fork repo ให้เป็นของตัวเอง**

กด Fork ที่มุมขวาบนของหน้า GitHub repo นี้ เพื่อคัดลอก repo มาไว้ใน account ของตัวเอง

**1.2 Clone repo ที่พึ่ง fork มา**

```bash
git clone https://github.com/<your-username>/<repo-name>.git
cd <repo-name>
```

**1.3 ตั้งค่าตัวตนใน repo (local config)**

```bash
git config --local user.name "Full name"
git config --local user.email "Your Email"
```

### 2. กรอกแฟ้มประวัติ `my-introduction.md`

เปิดแฟ้ม `my-introduction.md` แล้วบันทึกข้อมูลตัวตน:

- ชื่อรหัส (Codename)
- ความสามารถพิเศษ
- เสบียงประจำกาย
- สิ่งที่ต้องการฝึกปรือ
- หรืออะไรก็ได้ที่อยากให้หน่วยรู้จัก
